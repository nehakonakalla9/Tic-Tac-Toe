Tic Tac Toe Project
This is a simple Tic Tac Toe game implemented in Python. The game supports a customizable board size and can be extended to support multiple players in the future.

Features
Board Customization: You can set up an m * n board to play Tic Tac Toe.
Two Players: The game is currently designed for two players, where one plays as "X" and the other plays as "O".
Extendable: The project is built in an extensible way, allowing for future enhancements like supporting more than two players or adding AI players.


If you would like to contribute to this project, feel free to open a pull request. Bug reports and feature requests are also welcome.
